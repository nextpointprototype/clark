System Administration

NextPoint Group supports the technology infrastructure for mission critical systems by overseeing all operating system software and its successful integration with the hardware and applications software of major computing systems. We perform O&M activities by providing installation, patching, upgrades, backups, restore practices and deployments. Administrators are experienced in monitoring, diagnosing and resolving complex system and network problems.

Services Include:
  • Web and application servers such as Apache HTTPD, Apache Tomcat, WebLogic, WebSphere, or JBoss; Oracle, MySQL, and deploying and operating Linux (e.g. CentOS or Red Hat)
  • Deploying, automating and using Amazon Web Services (AWS) technologies, including AWSCLI, AWS SDKs, EC2, S3, CloudFormation, VPC, and Cloud Watch; assessing server loads and developing tuning recommendations:
  • DevOPS automation and deployment tools (e.g. Hudson, Jenkins, CruiseControl, Artifactory, Docker, Vagrant, Chef, Puppet, GitHub, Tenable Security Center, Jenkins, and Nexus.
Positions Include:
  • System Administer
  • LAN Support Specialist
  • Operations Engineer
